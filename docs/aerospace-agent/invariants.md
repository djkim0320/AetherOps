## Current implementation boundary

The A1/A2 pure-domain owners, policy-first tool routing and both A4 offline slices now have executable tests. These tests cover invariants 1-39 in their stated bounded domains and the no-certification/no-hardware-control catalog assertions. They do not establish persistence or UI coverage for every row.

`RunStateRevision` remains authoritative. `EngineeringRunState` is currently a deterministic projection and has not been cut over to worker-owned typed persistence. Memory lifecycle (40-43), independent review workflows (44), restricted export approval (47), aerospace-specific compute sandbox qualification (48), multi-owner work-order enforcement (49) and slice-specific restart completion (50) remain planned. A planned inspector location is not implementation evidence.

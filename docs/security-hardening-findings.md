## Deferred decision rules

- DNS connection pinning, browser profile/download lifecycle, graceful draining, and native OS secret protection require isolated integration work; they must not be represented as complete until their realistic boundary tests pass.
- Remote serving is not a documented product requirement. The safe compatibility-preserving decision for this pass is to keep loopback bootstrap unchanged and reject non-loopback binding.
